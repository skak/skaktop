Custom Request Headers
This is the list of changes made to outgoing request headers. When each switch is "on", Chrome will modify the header as you specify -- either by replacing the entire contents of the header, or by adding more text to it.

You can also use these tokens in your strings to be replaced with random values:
$RANDOM_NUMBER - inserts a random integer between 0 and 999999999999999.
$RANDOM_STRING - inserts a string of 10-50 random alphanumeric characters.
$RANDOM_IP - inserts a random IP address.


About

Header Hacker is an extension that allows you to modify HTTP request headers before they are sent to websites. You can either change headers on-the-fly with the icon in the upper-right, or set them to always change for a given site.

This extension is built and maintained by Glenn Wilson.

Permanent Switch List
This is the list of header changes that will always be made on the given site. Enter the desired hostname ("www.example.com") and select which header switch to make.
